# Blynks Backend

Backend-Repository für **Blynks**, eine Open-Source-Messaging-Plattform mit Fokus auf Einfachheit, Datenschutz und Erweiterbarkeit.

> ⚠️ Dieses Projekt befindet sich aktuell in einer **sehr frühen Entwicklungsphase**. Das Backend wird gerade erst aufgebaut und enthält derzeit noch keine Funktionalität.

---

## Über Blynks

Blynks ist ein Open-Source-Messaging-Ökosystem mit folgenden Zielen:

* **Datenschutzfreundlich**
* **Erweiterbar**
* **Modern und leichtgewichtig**

➡️ **Website Repository:**
https://github.com/jojosstudio/blynks

---

## Projektstatus

🚧 **Early Stage**

Dieses Repository bildet aktuell nur die **Grundlage für die Backend-Architektur**. In den nächsten Wochen und Monaten wird die Infrastruktur schrittweise aufgebaut.
